# 说明
复刻了pico轻世界里的伪3D按钮，利用了传送门的制作原理。传送门参考工程：https://github.com/SebLague/Portals 
- Unity版本2022.3.14f1c1
- 本项目没有VR相关的设置，使用键鼠模拟视野移动，有需要可自行集成
